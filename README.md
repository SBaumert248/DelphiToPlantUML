# DelphiToPlantUML
Convert Delphi / Pas Sourcecode to plant UML class diagramm.

## How to use
Convert Single File:

"python-path" "pas-file" "destination-file"

Convert all Files of one folder:

"python-path" "pas-folder" "destination-folder"


## How to read
| char  | description |
| ------------- |-------------|
| "+" | property |
| "-" | private |
| "#" | protected |
| "+" | public |
| "~" | published |
